# NickFury-s-heroManagementSystem
<p>
개발동기 및 목적
  
-Socket을 이용하여 멀티 네트워크 통신을 학습한다.

-자바의 Thread를 이용하여 원하는 기능을 구현한다.

-다양한 layout, panel등을 사용하여 원하는 gui의 모습을 구현한다.

-DB를 대체하여 HashMap으로 DB 학습 전, 사용에 대한 개념을 익힌다.

</p>
